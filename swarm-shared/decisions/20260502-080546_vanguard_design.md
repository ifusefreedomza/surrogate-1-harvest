# vanguard / design

### 1. **Diagnosis**
* The Vanguard project lacks a comprehensive solution to handle HF API rate limits, which can block dataset training and hinder the project's progress.
* The current implementation does not utilize the HF CDN bypass strategy, which can download public dataset files without being subject to the API rate limit.
* The project's design does not incorporate a mechanism to pre-list file paths and embed them in the training script, leading to potential rate limit issues during training.
* The existing codebase does not implement a studio reuse strategy, resulting in unnecessary recreation of studios and potential quota issues.
* The project's training pipeline is not optimized for performance, leading to potential bottlenecks and inefficiencies.

### 2. **Proposed change**
The proposed change will focus on implementing the HF CDN bypass strategy and pre-listing file paths to improve the project's performance and avoid rate limit issues. The changes will be made in the `train.py` file, specifically in the data loading and processing sections.

### 3. **Implementation**
To implement the proposed change, the following steps will be taken:
1. **Pre-list file paths**: Use the `list_repo_tree` API to pre-list the file paths for the dataset and save them to a JSON file.
2. **Embed file paths in training script**: Modify the `train.py` file to read the pre-listed file paths from the JSON file and use them to download the dataset files from the HF CDN.
3. **Implement HF CDN bypass**: Update the `train.py` file to use the HF CDN bypass strategy to download public dataset files without being subject to the API rate limit.

Example code snippet:
```python
import json
import requests

# Pre-list file paths
def pre_list_file_paths(repo_id, path):
    response = requests.get(f"https://huggingface.co/datasets/{repo_id}/tree/main/{path}")
    file_paths = response.json()
    with open("file_paths.json", "w") as f:
        json.dump(file_paths, f)

# Embed file paths in training script
def load_dataset(file_paths):
    dataset = []
    for file_path in file_paths:
        response = requests.get(f"https://huggingface.co/datasets/{repo_id}/resolve/main/{file_path}")
        dataset.append(response.content)
    return dataset

# Implement HF CDN bypass
def download_dataset(file_paths):
    dataset = []
    for file_path in file_paths:
        response = requests.get(f"https://huggingface.co/datasets/{repo_id}/resolve/main/{file_path}")
        dataset.append(response.content)
    return dataset
```
