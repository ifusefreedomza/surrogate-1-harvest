# surrogate-1 / design

### Diagnosis
* The project lacks a robust implementation for handling Hugging Face API rate limits, which can block dataset training.
* There is inadequate reuse of existing Lightning Studio instances, leading to wasted quota and potential downtime.
* The data ingestion pipeline is not optimized, relying heavily on the Hugging Face API with rate limits.
* The project does not have a mechanism to bypass the Hugging Face API rate limit for dataset training.
* The Lightning Studio instances are not properly stopped and restarted when idle, leading to training process deaths.

### Proposed change
The proposed change is to implement a mechanism to bypass the Hugging Face API rate limit for dataset training by using the HF CDN to download dataset files. This can be achieved by modifying the `train.py` script to use the `https://huggingface.co/datasets/{repo}/resolve/main/{path}` URL to download dataset files instead of using the Hugging Face API.

### Implementation
To implement this change, the following steps can be taken:
1. Modify the `train.py` script to use the HF CDN URL to download dataset files.
2. Use the `list_repo_tree` API call to get the list of files in the dataset repository and save it to a JSON file.
3. Embed the JSON file in the `train.py` script to use the CDN URLs to download the dataset files.
4. Use the `requests` library to download the dataset files from the CDN URLs.

Example code snippet:
```python
import requests
import json

# Get the list of files in the dataset repository
repo_tree = requests.get(f"https://huggingface.co/datasets/{repo}/tree/main").json()
file_list = [file["path"] for file in repo_tree["files"]]

# Save the file list to a JSON file
with open("file_list.json", "w") as f:
    json.dump(file_list, f)

# Embed the JSON file in the train.py script
with open("file_list.json", "r") as f:
    file_list = json.load(f)

# Use the CDN URLs to download the dataset files
for file in file_list:
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{file}"
    response = requests.get(url)
    with open(file, "wb") as f:
        f.write(response.content)
```
