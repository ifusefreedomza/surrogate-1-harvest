### Code Snippet
```python
import requests

# Define dataset repository and file path
repo = "dataset-repo"
file_path = "path/to/file.parquet"

# Download dataset file from HF CDN
url = f"https://huggingface.co/datasets/{repo}/resolve/main/{file_path}"
response = requests.get(url)

# Save dataset file to local disk
with open(file_path, "wb") as f:
    f.write(response.content)
```
This code snippet downloads a dataset file from the HF CDN using the `requests` library and saves it to the local disk. This approach avoids using the HF API and its associated rate limit.
