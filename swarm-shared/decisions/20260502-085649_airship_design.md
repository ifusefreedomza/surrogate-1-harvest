### Highest-Value Incremental Improvement
The highest-value incremental improvement that can ship in <2h is to implement the **HF CDN Bypass** pattern to avoid rate-limit blocks during dataset training. This involves downloading public dataset files directly from the HF CDN without using the HF API, which has a lower rate limit.

### Implementation Plan
1. **Update `train.py` script**:
	* Add a new function to download dataset files from the HF CDN using the `https://huggingface.co/datasets/{repo}/resolve/main/{path}` URL pattern.
	* Use the `requests` library to download the files.
2. **Pre-list file paths**:
	* Make a single API call to `list_repo_tree(path, recursive=False)` to get the list of file paths for a specific date folder.
	* Save the list of file paths to a JSON file.
3. **Embed file list in `train.py`**:
	* Load the JSON file containing the list of file paths in the `train.py` script.
	* Use the list of file paths to download the dataset files from the HF CDN.

### Code Snippets
```python
import requests
import json

# Download dataset files from HF CDN
def download_dataset_files(repo, path):
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"
    response = requests.get(url)
    with open(path, 'wb') as f:
        f.write(response.content)

# Pre-list file paths
def pre_list_file_paths(repo, date_folder):
    api_url = f"https://huggingface.co/api/v1/datasets/{repo}/tree?path={date_folder}&recursive=False"
    response = requests.get(api_url)
    file_paths = response.json()['files']
    with open('file_paths.json', 'w') as f:
        json.dump(file_paths, f)

# Embed file list in train.py
def load_file_paths():
    with open('file_paths.json', 'r') as f:
        file_paths = json.load(f)
    return file_paths

# Example usage
repo = "my-repo"
date_folder = "2026-04-29"
pre_list_file_paths(repo, date_folder)
file_paths = load_file_paths()
for file_path in file_paths:
    download_dataset_files(repo, file_path)
```
