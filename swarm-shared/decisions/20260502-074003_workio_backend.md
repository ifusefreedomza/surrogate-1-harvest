#### Step 1: Update `train.py` to Use HF CDN for Dataset Download
Modify the `train.py` script to download datasets from the HF CDN instead of using the HF API. This can be achieved by replacing the `load_dataset` function with a custom implementation that downloads the dataset files from the HF CDN.

```python
import requests

def download_dataset(repo_id, dataset_name):
    url = f"https://huggingface.co/datasets/{repo_id}/resolve/main/{dataset_name}"
    response = requests.get(url)
    with open(f"{dataset_name}.parquet", "wb") as f:
        f.write(response.content)

# Example usage:
download_dataset("axentx/workio-dataset", "train.parquet")
```

#### Step 3: Implement Studio Reuse and Idle Stop Handling
Implement studio reuse by checking if a studio with the same name and status is already running. If so, reuse the existing studio instead of creating a new one. Also, implement idle stop handling by checking the studio status before each `run` call and restarting the studio if it has stopped.

```python
import lightning

# Reuse existing studio if possible
for s in lightning.Teamspace.studios:
    if s.name == "workio-studio" and s.status == "Running":
        studio = s
        break
else:
    # Create new studio if no existing one is found
    studio = lightning.Studio(create_ok=True)

# Handle idle stop by checking studio status before each run call
if studio.status == "Stopped":
    studio.start(machine=lightning.Machine.L40S)
```
