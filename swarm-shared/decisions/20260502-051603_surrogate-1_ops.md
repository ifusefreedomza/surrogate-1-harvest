### Proposed change

### Implementation
To implement the HF CDN bypass, we will modify the `train.py` file to download dataset files from the HF CDN instead of using the Hugging Face API. We will also add a mechanism to reuse existing Lightning Studio instances.

```python
import requests
import json

# Define the HF CDN URL and dataset repository
cdn_url = "https://huggingface.co/datasets/{repo}/resolve/main/{path}"
dataset_repo = "axentx/surrogate-1"

# Define the Lightning Studio instance reuse mechanism
def reuse_studio(studio_name):
    for s in Teamspace.studios:
        if s.name == studio_name and s.status == 'Running':
            return s
    return None

# Download dataset files from the HF CDN
def download_dataset_files(dataset_path):
    file_list = requests.get(cdn_url.format(repo=dataset_repo, path=dataset_path)).json()
    for file in file_list:
        file_url = cdn_url.format(repo=dataset_repo, path=file['path'])
        response = requests.get(file_url)
        with open(file['path'], 'wb') as f:
            f.write(response.content)

# Reuse existing Lightning Studio instance or create a new one
studio = reuse_studio('surrogate-1')
if studio is None:
    studio = Teamspace.create_studio('surrogate-1')

# Start training with the reused or new studio instance
studio.start(machine=Machine.L40S)
```
