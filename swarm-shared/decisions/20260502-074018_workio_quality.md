#### Step 1: Update `train.py` to Use HF CDN Bypass
Modify the `train.py` script to download dataset files from the HF CDN instead of using the HF API. This will bypass the rate limit and allow for faster dataset training.

```python
import requests

# Define the dataset repository and file path
repo_id = "your-dataset-repo"
file_path = "path/to/dataset/file"

# Download the dataset file from the HF CDN
url = f"https://huggingface.co/datasets/{repo_id}/resolve/main/{file_path}"
response = requests.get(url)

# Save the dataset file to a local directory
with open(f"{file_path}", "wb") as f:
    f.write(response.content)
```

#### Step 2: Pre-List File Paths and Embed in Training Script
Make a single API call to `list_repo_tree` to get the list of file paths for the dataset repository. Save the list to a JSON file and embed it in the `train.py` script.

```python
import json

# Define the dataset repository and file path
repo_id = "your-dataset-repo"
file_path = "path/to/dataset/file"

# Get the list of file paths for the dataset repository
response = requests.get(f"https://huggingface.co/api/v1/datasets/{repo_id}/tree?path={file_path}&recursive=False")

# Save the list of file paths to a JSON file
with open("file_paths.json", "w") as f:
    json.dump(response.json(), f)

# Embed the list of file paths in the train.py script
with open("file_paths.json", "r") as f:
    file_paths = json.load(f)

# Use the list of file paths to download the dataset files from the HF CDN
for file_path in file_paths:
    url = f"https://huggingface.co/datasets/{repo_id}/resolve/main/{file_path}"
    response = requests.get(url)
    with open(file_path, "wb") as f:
        f.write(response.content)
```

#### Step 3: Implement Studio Reuse Instead of Recreate
Modify the `train.py` script to reuse an existing Lightning Studio instead of creating a new one. This will save 80hr/mo quota when iterating training scripts.

```python
import lightning

# Define the studio name and machine type
studio_name = "your-studio-name"
machine_type = "L40S"

# Get the list of existing studios
studios = lightning.Teamspace.studios()

# Reuse an existing studio if it exists
for studio in studios:
    if studio.name == studio_name and studio.status == "Running":
        studio = studio
        break
else:
    # Create a new studio if it doesn't exist
    studio = lightning.Studio.create(studio_name, machine_type)

# Use the studio to run the training script
studio.run()
```
